Questions

  1. The server in the client-server-architecture allows clients to
    request and receive services it offers.
  2. The client provide an interface to users of the system to
    request and receive services from the centralized server.
  3. An IP address is an identifier for each device connected to a
    network that uses the IP (Internet Protocol) for communication. No
    two devices connected to the same network may have the same IP
    address.
  4. DNS (Domain Name System) is a naming system for various
    resources connected to a network. It allows clients to connect to
    centralized servers by using a more memorable name instead of an IP
    address, because the DNS translates the domain name to the
    corresponding IP address.
  5. The Internet is a global networks of networks, and the World
    Wide Web is the collection of information accessible through the
    Internet.
  6. HTML (Hyper Text Markup Language) is the standard markup
    language for Web pages, and describes the structure of said Web
    pages.
